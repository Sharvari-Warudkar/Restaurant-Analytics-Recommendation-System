# Restaurant Analytics & Recommendation System

## Project Overview

This project analyzes restaurant data to identify high-performing restaurants, customer preferences, cuisine trends, and factors influencing ratings. The goal is to generate business recommendations for a restaurant consolidator platform.

---

## Business Problem

A restaurant consolidator wants to improve its B2C platform by identifying star restaurants and generating recommendations using customer ratings, votes, cuisines, location data, delivery options, and booking facilities.

---

## Dataset

Dataset contains:

- Restaurant Details
- Country Codes
- Customer Ratings
- Votes
- Cuisines
- Delivery Information
- Table Booking Information

---

## Tools Used

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- SQL
- Tableau

---

## Project Workflow

### Data Cleaning

- Removed duplicate records
- Handled missing values
- Validated data consistency

### Exploratory Data Analysis

- Restaurant distribution by city
- Top cuisines analysis
- Cost distribution analysis
- Rating distribution
- Delivery vs Non-delivery analysis
- Table booking analysis

### Business Insights

- Identified cities with highest restaurant concentration
- Determined most popular cuisines
- Analyzed impact of delivery on ratings
- Identified top-performing restaurants

---

## Key Findings

✅ Top restaurants generated maximum customer engagement through higher ratings and votes.

✅ Restaurants with delivery options received significantly higher customer attention.

✅ A limited number of cuisines contributed to majority customer demand.

✅ Major restaurant concentration was found in a few specific cities.

---

## Business Impact

- Improved restaurant recommendation strategy
- Enhanced customer targeting
- Identified high-performing restaurant segments
- Supported data-driven expansion decisions
